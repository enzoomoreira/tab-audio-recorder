    Settings

    Recording

    Higher bitrate = larger files, better quality. Affects capture from HTML <audio> / <video> elements and is reused as the MP3 export bitrate. Network streams (e.g. radio.garden) are captured at source quality regardless.

    0 = unlimited. Recordings are held in memory until you stop them, so a cap protects against runaway memory use on long streams. The recording auto-stops and is saved when the limit is reached.

    Export

    Recordings are captured as WebM/Opus and converted on export. WAV is lossless (larger files); MP3 is smaller and uses the bitrate above for its quality.

    Files are saved to Downloads/<subfolder>/. Cannot escape the Downloads directory (browser security).

    Variables: {host}, {title}, {date}, {time}, {timestamp}. Extension is added automatically.
