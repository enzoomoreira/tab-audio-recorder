Settings

Recording

Higher bitrate = larger files, better quality. Only affects sites with HTML <audio> / <video> elements. Network streams (e.g. radio.garden) are recorded at source quality regardless.

0 = unlimited. Recordings are held in memory until you stop them, so a cap protects against runaway memory use on long streams. The recording auto-stops and is saved when the limit is reached.

Export

Files are saved to Downloads/<subfolder>/. Cannot escape the Downloads directory (browser security).

Variables: {host}, {title}, {date}, {time}, {timestamp}. Extension is added automatically.
