13 - <audio> with faked mediaKeys (P4 check)

Expected outcome: recording refused with "DRM/EME content cannot be captured"

We stub element.mediaKeys via Object.defineProperty so the P4 check in StreamRecorder treats this element as if it were EME-protected. Real DRM (Spotify, Apple Music) would set this via setMediaKeys(); the check in our code is a simple != null guard.
